Economic Software System "PREMIER"

Premiere is an economic software system that was designed especially for small and medium businesses.

The Tallium team was asked to create a new, more modern version of the software made for mobile devises.

We studied the details thoroughly, added a little bit of magic — and made this!

Invoice creation

it is easy to generate invoices (both simple and complex).

What exactly did we do?

We created a software that can be easily used by both experienced users and beginners.
Here’s the list of features that make new program totally amazing:

  • New design
  • User-friendly
    interface
  • Simple usage
  • Online version

How did we do this?

We used:

  • Backbone.js, Require.js and Underscore.js for front-end
  • PHP, Laravel, PayPal SDK, JWT Auth and synchronization with Windows applications for back-end
  • CSS3 and HTML5 for layout

Convenient PDF file generation:

now you can easily print your invoices
or send them via email.
